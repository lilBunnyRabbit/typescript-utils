Class EventEmitter<TEvents>

Event emitter class capable of emitting and listening for typed events. It enables the creation of a strongly-typed, custom event handling system.

Example

type MyEvents = {
  data: string;
  loaded: void;
  error: Error;
}

const emitter = new EventEmitter<MyEvents>();

Type Parameters

  • TEvents extends Record<PropertyKey, unknown>

    An object type mapping event names to their associated data types. Each key represents an event name, and its value represents the type of data associated with that event.
